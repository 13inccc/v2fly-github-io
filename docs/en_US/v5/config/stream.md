# Stream

```json
{
  "transport":"tcp",
  "transportSettings":{},
  "security":"none",
  "securitySettings":{}
}
```

> `transport`: name of <transport>

> `transportSettings`: settings of <transport>

> `security`: name of <security>

> `securitySettings`: settings of <security>

> `socketSettings`: [SocketConfigObject](#SocketConfigObject)
